# BossBarAPI-Nukkit
BossBarAPI是为其他插件提供Boss血条API接口便捷服务
BossBarAPI provide convenient service for other plug-ins generated bossbar
- 简单实用
- 接口共**5个**，分别是(***addBossBar***,***sendBossBarToPlayer***,***setTitle***,***removeBossBar***,***playerMove***)
